Hệ thống Cloud Mini 2026

Nền tảng định hình tương lai mây ảo, quy tụ 9 server quyền lực cấu trúc Microservices chạy trên Docker containers.

Hệ Sinh Thái Quyền Lực

language

Web Server

Nginx phục vụ giao diện web tĩnh với hiệu suất cao và độ trễ cực thấp, tích hợp Load Balancing.

terminal

App Server

Flask backend xử lý API requests và business logic phức tạp với RESTful endpoints.

database

Database

MariaDB lưu trữ dữ liệu quan hệ an toàn, bền bỉ với khả năng backup tự động.

fingerprint

Authentication

Keycloak quản lý xác thực tập trung và phân quyền chuẩn Enterprise SSO / OIDC.

cloud_queue

Object Storage

MinIO thay thế Amazon S3, kho lưu trữ phân tán cho hình ảnh và tài liệu.

dns

DNS Server

Bind9 phân giải tên miền nội bộ cloud.local với độ trễ cực thấp.

analytics

Monitoring

Prometheus thu thập metrics giám sát từ tất cả services theo thời gian thực 24/7.

dashboard

Dashboard

Grafana hiển thị biểu đồ giám sát realtime trực quan và mãn nhãn.

router

API Gateway

Nginx reverse proxy định tuyến API và cân bằng tải cho web cluster.
